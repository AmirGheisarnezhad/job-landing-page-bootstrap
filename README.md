# 💼 Job Landing Page

Questo progetto è una landing page responsive sviluppata con Bootstrap 5.

## 🚀 Demo

👉 https://amirgheisarnezhad.github.io/job-landing-page-bootstrap/

## 🛠️ Tecnologie utilizzate

* HTML5
* CSS3
* Bootstrap 5
* JavaScript

## 📌 Funzionalità

* Navbar responsive
* Hero section con immagine di sfondo
* Form di login con validazione
* Slider con JavaScript
* Layout responsive per mobile e desktop

## 🎯 Obiettivo

Questo progetto è stato realizzato per esercitarmi nello sviluppo front-end e migliorare le mie competenze pratiche.
